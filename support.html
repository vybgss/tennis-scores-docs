Frequently Asked Questions (FAQ)

  • 1. How do I start tracking a match?
    Simply enter the players’ names and tap "Start Match." Once the match is started, you can record points as they happen.
  • 2. How do I add a point to a player?
    During an active match, tap the "Point to [Player]" button for the player who won the point. The score updates automatically.
  • 3. Can I track sets and tiebreaks?
    Yes. The app automatically handles scoring progression through games, sets, and tiebreaks. When a set reaches 6–6, a tiebreak is initiated and scoring adjusts accordingly.
  • 4. What if I need to reset or end the match early?
    Tap "End Match" to confirm ending the current match. You can start a new one afterwards. If you need to completely reset, use the "Reset" option.
  • 5. Can I change player names mid-match?
    Player names should be set before starting the match. To change them, end or reset the current match, update the names, then start a new match.
